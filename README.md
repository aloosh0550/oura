# 🧠 عليOS - لوحة القيادة الذكية

**نسخة مبسطة سهلة الفهم:**

## ايش هالمشروع؟

**عليOS** هي لوحة تحكم شاملة لإدارة حياتك (شخصية + عمل) في مكان واحد. تساعدك تنظم كل شيء وتركز على الأشياء اللي تهمك.

---

## 🎯 الميزات الرئيسية

### 1️⃣ **المهام والمشاريع**
- أضف مهامك اليومية وتابع تقدمها
- رتب المهام حسب الأولوية (عالي/متوسط/منخفض)
- قسمها لفئات: عمل 💼، شخصي 🎯، عادات 🔥
- شُف المشاريع وتقدمها وكم الإيراد المتوقع

### 2️⃣ **العادات والصحة**
- تتبع عاداتك اليومية (رياضة، قراءة، نوم...)
- رح تشوف رسوم بيانية توضح كيفية تطورك
- نصائح ذكية بناء على نومك والعادات

### 3️⃣ **إدارة المالية**
- تابع الديون والإيرادات والمصاريف
- اختر بين الريال السعودي 🇸🇦 أو الدولار 💵
- شُف كل معاملاتك في جدول واضح

### 4️⃣ **مؤقت التركيز**
- مؤقت Pomodoro (25 دقيقة تركيز + استراحة)
- في السايد بار يظهر الوقت الباقي
- يحولك بين وضع التركيز والاستراحة

### 5️⃣ **التقارير الذكية**
- تقرير AI شامل لأسبوعك كامل
- يحللك أدائك وإنجازاتك
- بيديك insights وتوصيات

### 6️⃣ **وضعين للاستخدام**
- **🌿 الوضع الشخصي**: للحياة اليومية
- **💼 الوضع التجاري**: للعمل والمشاريع

---

## 🛠 كيفية الاستخدام

1. **اتدخل على الموقع** → كل حاجتك محفوظة عندك محليا (LocalStorage)
2. **اختار الوضع اللي يناسبك** (شخصي أو تجاري)
3. **أضف مهامك** باستخدام زر `+` في الأعلى
4. **تابع تقدمك** من لوحة التحكم الرئيسية
5. **اضبط طاقتك** وسجل عاداتك يوميا
